# SVD разложение для видео

Задача: удалить фон / оставить только фон видео.

Что использую: библиотека numpy, а также вспомогательные библиотеки для работы с видео(cv2, cv2_imshow импортируем отдельно тк находимся в гугл колаб), группирую кадры видео по каналам, составляя матрицы каналов, затем делаю SVD разложение матриц и изменяю элементы вектора сигнатурных чисел.

Результаты: получено два новых видео. Одно из них - это статичная картинка, другое - видео без фона.
